Installation

Install Fluenth into your Hono project using your preferred package manager. Fluenth has minimal dependencies and works across all Hono-supported runtimes.

Package Managers

Choose your preferred package manager:

npm
$ npm install fluenth
pnpm
$ pnpm add fluenth
bun
$ bun add fluenth

Peer Dependencies

Fluenth requires hono and zod as peer dependencies. If you don't have them:

$ npm install hono zod fluenth

TypeScript Configuration

Fluenth ships with full TypeScript declarations. Ensure your tsconfig.json has at least:

tsconfig.json
json 
{
  "compilerOptions": {
    "target": "ES2022",
    "module": "NodeNext",
    "moduleResolution": "NodeNext",
    "strict": true,
    "exactOptionalPropertyTypes": true
  }
}

Runtime Support

Runtime Status Notes
Node.js ≥ 18 Supported Recommended for local dev
Bun Supported Fastest local dev experience
Deno Supported Use npm: prefix
Cloudflare Workers Supported Edge-compatible
AWS Lambda Compatible Via Hono adapter
Vercel Edge Compatible Via Hono adapter

Verify Installation

After installing, verify by importing from fluenth:

test-import.ts
typescript 
import { fluenth, describe } from 'fluenth'

// Should resolve without errors
console.log('Fluenth installed correctly ✓')
Warning

If you see a peer dependency warning about hono, ensure your Hono version is >= 4.0.0. Older versions of Hono had a different middleware signature that Fluenth does not support.
